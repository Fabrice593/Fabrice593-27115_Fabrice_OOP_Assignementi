# Java Management Systems - Complete Collection

## About This Project
A comprehensive suite of 6 Java console applications that demonstrate real-world management systems with full input validation and data processing capabilities.

## Applications Included

### 1. Stock Management System
**Core Functionality**: Inventory tracking and product management
- **Warehouse Management**: Track multiple storage locations
- **Supplier Database**: Maintain vendor information


### 2. Flight Booking System
**Core Functionality**: Airline reservation management
- **Flight Operations**: Manage schedules and routes
- **Passenger Management**: Handle customer bookings
- **Crew Assignment**: Assign pilots and flight attendants


### 3. Tax Administration System
**Core Functionality**: Revenue collection and taxpayer management
- **Taxpayer Registration**: Maintain citizen tax records
- **Employer Reporting**: Process employer submissions


### 4. Health Center System
**Core Functionality**: Medical records and appointment management
- **Patient Profiles**: Maintain medical histories
- **Appointment Scheduling**: Manage doctor visits
- **Medical Staff Management**: Track healthcare providers


### 5. Government Service System
**Core Functionality**: Civil documentation and citizen services
- **ID Processing**: Handle national identification
- **Certificate Management**: Issue birth/death certificates
- **Citizen Database**: Maintain population records


### 6. Student Management System
**Core Functionality**: Academic administration
- **Student Records**: Maintain personal and academic data
- **Course Management**: Handle subject offerings
- **Grade Tracking**: Record academic performance


## Validation Features

### Data Integrity Checks
- **Numeric Validation**: Range checking and positive number enforcement
- **Date Validation**: Format verification and logical date sequencing
- **String Validation**: Length checks and format patterns


### Business Logic Validation
- **Relationship Checks**: Cross-reference validation between entities
- **Consistency Verification**: Data coherence across related fields
